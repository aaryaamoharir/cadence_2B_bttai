# Cadence_2B
Breakthrough Tech AI Studio Project Fall 2025
1) Set up a virtual enviroment with `python -m venv myenv`
2) Then call `myenv\Scripts\activate` to activate the virtual enviroment
3) Now, install requirements: `pip install -r requirements.txt`
4) Lastly, run the command `streamlit run dashboard.py` in your terminal to run it
